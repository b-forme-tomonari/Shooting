# シューティングゲーム
## 概要
Kotlinを使用したAndroidで動作するシューティングゲームです。

## 機能
指のスライドで機体を操作することができ、タップ中は弾が発射される。

弾が敵に当たるとダメージを与えることができて、敵のHP以上のダメージを与えると消滅する。

## 採用技術一覧
採用技術は以下の通りです。
|概要|詳細|
|:--|:--|
| プログラミング言語 | Kotlin（ver:1.7.10）|
| 統合開発環境 | Android Studio（ver:2022.2.1 Patch 2）|
